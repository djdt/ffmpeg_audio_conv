# ffmpeg_audio_conv

Multi thread audio converter script using ffmpeg.

Usage:
```
audio_conv.py [-h] [-b BITRATE] [-c COPYEXTS [COPYEXTS ...]] [-C]
                     [-i INEXTS [INEXTS ...]] [-p] -o OUTEXT [-q QUALITY]
                     [-t THREADS] [-v]
                     indir outdir
```
